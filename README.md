# 6-Day Meal Plan (GitHub Pages)

Simple static app (HTML/CSS/JS) showing:
- 6-day lunch + dinner plan
- Sunday braai guidelines
- Shopping list with tick boxes (saved in localStorage)

## Host on GitHub Pages
1. Create a new GitHub repo (e.g. `meal-plan-app`)
2. Add these files to the repo root:
   - index.html
   - style.css
   - app.js
